# Suporte — LegisKids

## Sobre o Projeto

O **LegisKids** é um projeto acadêmico desenvolvido na disciplina de Métodos de Desenvolvimento de Software (MDS) da FCTE/UnB.

---

## Como Obter Ajuda

Caso precise de suporte:

* abra uma [issue](../../issues) no repositório
* descreva claramente o problema
* informe passos para reprodução
* inclua prints ou logs, se necessário

---

## Antes de Abrir uma Issue

Verifique se:

* o problema já foi reportado
* a documentação foi consultada
* a dúvida não está respondida em outra issue ou Pull Request

---

## Tipos de Solicitação

### Bugs

Inclua:

* comportamento esperado
* comportamento atual
* passos para reproduzir
* ambiente utilizado

---

### Dúvidas

Informe:

* contexto do problema
* trecho relacionado
* tentativa realizada anteriormente

---

### Sugestões

Contribuições e melhorias são bem-vindas.

Ao sugerir mudanças:

* explique o problema atual
* apresente a proposta
* descreva benefícios esperados

---

## Tempo de Resposta

Como este é um projeto acadêmico, o tempo de resposta pode variar conforme a disponibilidade da equipe.
